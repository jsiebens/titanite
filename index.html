Titanite

A small web framework for Java 8, inspired by others like Sinatra and Finatra, running on top of Netty

highlights

  • Async with Java 8 CompletableFuture
  • Running on top of Netty
  • Templating with Mustache or Freemarker
  • JSON with Jackson or Gson
  • File upload
  • Serving static resources
  • ...
